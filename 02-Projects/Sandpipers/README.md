# Sandpipers of Nevada

## Status
**Schedule rebuild in progress**

## Overview
Coaching schedule management for Sandpipers of Nevada swim team. Rebuilding the high school season schedule in Google Sheets using BCA South as a template.

## Key Files (Google Drive)
- BCA South Site Coaching Schedule — the "good" template to replicate
- 2026 High School Season Schedule — source data to restructure
- 2026_HS_Season_Schedule_Structured — target output file

## Target Structure
1. Base Schedule tab (template)
2. Base Lanes tab (template)
3. Weekly Schedule + Lanes tabs as changes happen
4. Past-weeks compilation tab

## BCA Pattern
- `M.DD.26 Lanes` = planner view
- `M.DD.26 Schedule` = coach view
- Paired per week
- Groups: ES Wahoo, ES Dolphin, Wahoo A/B/C, Dolphin A/B, Marlin A/B, BEACH 1-5, Senior 1-4 AM/PM, High Performance AM/PM, Intro to Pre Comp

## End State
- Live Google Sheet shared with coaches for daily viewing
- Parents eventually get Google Calendar links (later phase)
